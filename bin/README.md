# bin - Launchers and Installers

## Quick Launchers

| File | Action |
|------|--------|
| qwen.bat | Start Qwen safe workspace |
| safe-workspace.bat | Direct safe workspace launch |
| aider-lmstudio.bat | Aider with LM Studio |
| optimize-as-admin.bat | System optimization |

## PowerShell Scripts

| File | Action |
|------|--------|
| install-profile.ps1 | Install PowerShell commands |
| install-powershell.ps1 | PowerShell setup |

## Usage

After running install-profile.ps1, use these from ANY PowerShell:

    qwen              # Start workspace
    q                 # Short alias
    qwen-backup       # Backup utilities
    qwen-undo         # Undo last change
    qwen-help         # Show help
