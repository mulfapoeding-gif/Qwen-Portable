# Projects - Your AI Work

## Default Save Location

All your AI projects should be saved here. This is the only place
AI agents should create files.

## Structure

    projects/
    ├── sandbox/          # Testing and experiments
    ├── project-1/        # Your actual projects
    └── project-2/

## Safety

- Git auto-commit enabled in sandbox
- Automatic backups before changes
- Protected directories enforced

## Sync to USB

Active projects are automatically synced to USB backup.
