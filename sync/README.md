# sync - USB Backup and Sync

## Purpose

Keep your USB backup synchronized with your PC installation.

## Usage

### Backup to USB

1. Plug in 32GB USB drive
2. Run sync-to-usb.bat
3. Wait for completion

### Restore from USB

1. Plug in USB drive
2. Run sync-from-usb.bat
3. Confirm restore operation

## What Gets Synced

- [OK] Code and tools
- [OK] Configuration files
- [OK] Active projects
- [OK] Documentation
- [OK] 1 core model (Qwen2.5-Coder-3B)
- [SKIP] Large model files (except core)
- [SKIP] Logs and temp files

See ../docs/USB-BACKUP-GUIDE.md for details.
