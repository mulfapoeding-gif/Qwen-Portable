## 🐛 Troubleshooting

### "Command not recognized"
```powershell
# Reload profile
. $PROFILE

# Or restart PowerShell
```

### "No results"
- Check internet connection
- Try different keywords
- DuckDuckGo may be temporarily unavailable

### "Warning about ddgs package"
- Just a rename notice, still works
- Can ignore or run: `pip install ddgs`
