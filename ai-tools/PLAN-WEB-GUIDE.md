## ✅ New Mode Added!

### Updated Toggle:
```
🟡 PLAN       →  🟢 PLAN-WEB  →  🔵 AUTO  →  🔴 YOLO
```

---

## 🎯 What Plan-Web Does:

When you use **PLAN-WEB** mode:

1. **🔍 Researches** your topic on the internet
2. **📚 Finds** best practices, tutorials, examples
3. **🌐 Shows** sources and references
4. **📋 Creates** informed plan based on research
5. **✅ Waits** for your approval before proceeding

---

## 🚀 How to Use:

### In Qwen Session:

```
>>> TAB  (until you see 🟢 PLAN-WEB)

>>> build a python web scraper

🌐 PLAN-WEB Mode Active

🔍 Researching: "build a python web scraper best practices 2025"
📚 Finding: Tutorials, documentation, examples

══════════════════════════════════════════════════════════
Research Results:

1. Real Python - BeautifulSoup Tutorial
   Source: https://realpython.com/beautiful-soup/
   Learn to parse HTML and extract data

2. BeautifulSoup Documentation
   Source: https://www.crummy.com/software/BeautifulSoup/
   Official docs with examples

3. StackOverflow - Web Scraping Questions
   Source: https://stackoverflow.com/questions/tagged/web-scraping
   Common issues and solutions

══════════════════════════════════════════════════════════

📋 Informed Plan:

Task 1: Research (30 min)
  - Study BeautifulSoup docs
  - Review example code

Ready to proceed? [Y/n]
```

## 🐛 Troubleshooting:

### "No results found"
- Check internet connection
- Try more specific query
- DuckDuckGo may be temporarily unavailable

### "Research taking long"
- Web searches take 10-30 seconds
- Multiple searches are performed
- Wait for complete results
